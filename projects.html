Monopoly Web App [BETA]
In order to introduce data handling to implement different economic systems into Hasbro's classic game Monopoly, we have developed a web app powered by JavaScript and displayed with HTML and CSS. Our goal is to gain an insight into the dynamics of different economic systems as they relate to the game, enhance gameplay, and provide an easy to use interface to play with friends.

Visit the GitHub page to learn more and browse source code.

Creatures [ALPHA]
Simulating living creatures is no small feat - but allows us to discover trends and patterns in the behavior and evolution of organisms. Coded in Java, each creature has stats for many traits like health, aggression, attractiveness, strength, and curiosity. Each trait will dynamically affect how creatures interact with each other and the world around them.

Visit the GitHub page to learn more and browse source code.
